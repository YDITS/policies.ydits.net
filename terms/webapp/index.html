YDITS for Web 利用規約

第１条　概要

この「YDITS for Web 利用規約」は，よね/Yoneが提供するサービス「YDITS」で公開する「YDITS for Web」の利用条件を、本サイトを利用されるすべての方との間で定めるものです。

第２条　定義

  1. 本規約

    この「YDITS for Web 利用規約」を指します。

  2. 本サービス

    YDITSを指します。

  3. 本サイト

    YDITS for Webを指します。

  4. 当方

    本サービスを提供する「よね/Yone」を指します。

  5. ユーザー

    本サイトを利用されるすべての方を指します。

第３条　適用範囲

  1. ユーザーは，本サービスを利用するには本規約に同意しなければなりません。

  2. 本規約は，ユーザーが本サービスを利用した時点で適用されるものとします。

  3. ユーザーは，本規約の定めにしたがって本サービスを利用するものとします。

第４条　禁止事項

  1. 当方は、ユーザーが本サイトを利用するにあたって以下の行為を禁止とします。

    1. 法令，裁判所の判決もしくは命令，または法令上拘束力のある行政措置に反する行為
    2. 当方及び第三者の著作権，商標権，特許権等の知的財産権，名誉権，プライバシー権，肖像権，人格権，またはその他法令上または契約上の権利を侵害する行為
    3. 公序良俗に反する行為
    4. 本サイトで利用される当方及びその他のサーバーに支障を与える行為，またはそのおそれがある行為
    5. 本サイトに関して，第三者の機関に問い合わせる行為
    6. 本サイトに含まれる情報を，追加，改変，削除して虚偽の内容を作成する行為
    7. 本サイトの提供を妨害する行為，またはそのおそれがある行為

第５条　免責

  1. 当方は，本サービスの利用によって生じた損害の責任は一切負わないものとします。
  2. 当方は，ユーザーに事前に通知することなく，本サービスの一部または全部の内容を変更または停止できるものとし，それに伴うユーザーへのいかなる損害の責任は一切負わないものとします。

第６条　不保証

  1. 当方は，本サイトの情報の正確性，信頼性，目的適合性，エラー，バグの問題，その他一切の事項について保証しないものとします。

第７条　連絡方法

  1. ユーザーは，本サイトに関する連絡は，当方が指定する お問い合わせフォーム open_in_new にて行うものとします。

第８条　本規約の変更

  1. 当方は，ユーザーに個別の同意を得ることなく，本規約を変更することができるものとします。

  2. 当方は，本規約の変更時に，その内容について以下のウェブサイトの一つ以上に事前に表示し, ユーザーに通知します。

    1. 本サービスのTwitterアカウント(@YDITS_JP) open_in_new にてお知らせします。
    2. 本サービスのお知らせページ open_in_new にてお知らせします。

第９条　分離可能性

  1. 本規約の一部が消費者保護法及びその他の法令により無効とされる場合でも，その他の部分はこれに反しない最大限の範囲で効力を有するものとします。

第１０条　準拠法

  1. 本規約の正文は日本語とし，日本法において解釈されるものとします。

  2. 本サイトを利用したことに起因する、ユーザーと当方との間に生じた一切の紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
